# 법령 데이터 분석 체크리스트

## 조항별 분석 (15호 각각)

### 필수 항목

- [ ] 금지 행위 1줄 요약
- [ ] 탐지 난이도 분류 (keyword / context / complex)
- [ ] 핵심 키워드 3~5개
- [ ] 정규식 패턴 작성
- [ ] 위반 예시 2~3개
- [ ] 비위반 예시 1~2개
- [ ] 벌칙 정보

### 탐지 난이도 분류 기준

| 난이도 | 설명 | 판단 주체 |
|--------|------|----------|
| `keyword` | 특정 단어/패턴 매칭으로 탐지 가능 | 1단계 정규식 스캔 |
| `context` | 문맥 분석 필요 | Haiku 분류 |
| `complex` | 복합 판단 또는 외부 검증 필요 | Sonnet 정밀 판단 |

## 시술별 분석

### 필수 항목

- [ ] 공식 명칭
- [ ] 별칭(aliases) 목록
- [ ] 진료과목
- [ ] 필수 고지 부작용
- [ ] 흔한 위반 유형 (어떤 호 위반?)
- [ ] 특별 규제사항

### 시술별 필수 고지 부작용 참조

| 시술 | 필수 부작용 |
|------|-----------|
| 보톡스 | 멍, 비대칭, 두통, 안검하수 |
| 필러 | 멍, 부기, 혈관 폐색, 괴사 |
| 리프팅 | 통증, 멍, 감각 이상, 비대칭 |
| 지방흡입 | 출혈, 감염, 피부 울퉁불퉁, 혈전 |
| 레이저 | 화상, 색소침착, 흉터 |
| 임플란트 | 감염, 골유착 실패, 신경 손상 |

## 키워드 추출 원칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| 법령 용어 우선 | 법령에 명시된 표현 사용 | "치료 경험담" (O), "후기" (△) |
| 오탐 방지 | 일반 단어는 복합 패턴으로 | "효과" 단독 X → "효과\s*(봤|있었)" |
| 정규식 호환 | 특수문자 이스케이프 | `%` → `\\%` |
| 별칭 포함 | 동의어/브랜드명 포함 | 보톡스, 보툴리눔, 나보타 |
