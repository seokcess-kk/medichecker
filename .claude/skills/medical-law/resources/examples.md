## 제2호: 치료경험담

### 위반 예시

#### 2-V-001: 환자 후기 인용
```
"저는 00피부과에서 보톡스 시술을 받았는데, 정말 주름이 싹 없어졌어요!
5년은 젊어 보인다는 말을 들었습니다. 강력 추천합니다!"
- 위반 조항: 제56조 제2항 제2호
- 위반 사유: 환자 치료경험담으로 치료 효과 오인 유발
- 확신도: 90%
```

#### 2-V-002: 전후사진 비교
```
"보톡스 시술 전후 사진입니다.
Before: 이마 주름이 깊게 패인 모습
After: 매끈하고 탱탱해진 피부
놀라운 변화를 직접 확인하세요!"
- 위반 조항: 제56조 제2항 제2호
- 위반 사유: 전후사진 비교로 치료 효과 오인 유발
- 확신도: 95%
```

#### 2-V-003: 6개월 이하 임상경력
```
"개원 3개월 만에 보톡스 시술 500건 돌파!
믿고 맡기세요."
- 위반 조항: 제56조 제2항 제2호
- 위반 사유: 6개월 이하 임상경력 광고 (시행령 제23조 제1항 제2호)
- 확신도: 95%
```

### 비위반 예시

#### 2-NV-001: 시설/서비스 후기
```
"상담이 친절하고 대기 시간이 짧았어요.
주차 공간도 넓어서 편리했습니다."
- 비위반 사유: 치료 효과와 무관한 서비스/시설 후기
```

#### 2-NV-002: 일반적 시술 안내
```
"보톡스는 표정 주름 개선에 사용되는 시술입니다.
시술 시간은 약 10-15분 소요됩니다."
- 비위반 사유: 객관적 시술 정보 제공
```

## 제3호: 거짓 광고

### 위반 예시

#### 3-V-001: 100% 성공률
```
"저희 병원 보톡스 시술은 100% 성공률을 자랑합니다.
실패 없는 시술, 00클리닉에서 경험하세요."
- 위반 조항: 제56조 제2항 제3호
- 위반 사유: 검증 불가능한 100% 성공률 주장
- 확신도: 95%
```

#### 3-V-002: 부작용 없음 주장
```
"필러 시술, 부작용 걱정 없이 안심하세요!
저희 병원은 부작용이 전혀 없습니다."
- 위반 조항: 제56조 제2항 제3호
- 위반 사유: 의학적으로 거짓인 부작용 없음 주장
- 확신도: 95%
```

### 비위반 예시

#### 3-NV-001: 검증 가능한 정보

```
"OO대학교병원 출신 전문의
10년 이상의 시술 경험"
- 비위반 사유: 검증 가능한 사실 기재
```

## 제7호: 부작용 정보 누락

### 위반 예시

#### 7-V-001: 부작용 미고지
```
"보톡스 시술 - 간단하고 안전한 10분 시술!
점심시간에 받고 바로 일상으로 돌아가세요."
- 위반 조항: 제56조 제2항 제7호
- 위반 사유: 보톡스 필수 부작용(멍, 두통, 비대칭) 미고지
- 확신도: 85%
```

#### 7-V-002: 다운타임 은폐
```
"필러 시술 후 다운타임 제로!
바로 일상 복귀 가능합니다."
- 위반 조항: 제56조 제2항 제7호
- 위반 사유: 필러 필수 부작용(멍, 붓기) 미고지, 다운타임 허위
- 확신도: 90%
```

### 비위반 예시

#### 7-NV-001: 부작용 고지
```
"보톡스 시술 안내
- 시술 시간: 약 10-15분
- 주의사항: 시술 후 멍, 붓기, 두통이 나타날 수 있습니다.
- 드물게 비대칭이나 안검하수가 발생할 수 있으며, 시술 전 상담이 필요합니다."
- 비위반 사유: 필수 부작용 적절히 고지
```

## 복합 위반 사례

### CV-001: 2호 + 7호 + 8호 복합
```
"저희 병원에서 보톡스 시술 받으신 분들의 후기입니다.
'주름이 완전히 사라졌어요! 10년은 젊어 보인대요. 강추!'
보톡스 시술, 간단하고 안전한 10분 시술!
기적 같은 변화를 경험하세요."

- 위반 조항:
  1. 제56조 제2항 제2호 (치료경험담)
  2. 제56조 제2항 제7호 (부작용 미고지)
  3. 제56조 제2항 제8호 (기적 - 과장)
- 확신도: 2호 90%, 7호 85%, 8호 85%
```

### CV-002: 3호 + 4호 + 8호 복합
```
"다른 병원보다 훨씬 자연스러운 결과!
저희 보톡스는 100% 성공률, 업계 유일한 기술력으로
완벽한 결과를 보장합니다."

- 위반 조항:
  1. 제56조 제2항 제3호 (100% 성공률 - 거짓)
  2. 제56조 제2항 제4호 (다른 병원보다 - 비교)
  3. 제56조 제2항 제8호 (업계 유일, 완벽 - 과장)
- 확신도: 3호 95%, 4호 90%, 8호 85%
```

## 경계 사례 (판단 어려움)

### B-001: 암묵적 비교
```
"저희만의 특별한 노하우로 차별화된 결과를 드립니다."
- 판단: 직접적 비교는 아니나 "차별화"가 비교 암시
- 확신도: 55% (낮음)
- 검토 필요: 맥락에 따라 위반 여부 달라질 수 있음
```

### B-002: 일반적 표현
```
"최선의 결과를 위해 노력하겠습니다."
- 판단: "최선"은 일반적 표현으로 과장으로 보기 어려움
- 확신도: 20% (비위반 추정)
```

### B-003: 효과 암시
```
"많은 분들이 만족하셨습니다."
- 판단: 직접적 후기는 아니나 효과 암시 가능성
- 확신도: 60% (경계)
- 검토 필요: 구체적 맥락 확인 필요
```
