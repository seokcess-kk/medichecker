# 의료광고 위반 유형 상세

> 의료법 제56조 제2항 각 호별 상세 분석

---

## 탐지 난이도 분류 기준

| 난이도 | 설명 | 판단 주체 |
|--------|------|----------|
| `keyword` | 특정 단어/패턴 매칭으로 탐지 가능 | 1단계 정규식 스캔 |
| `context` | 문맥 분석 필요 | Haiku 분류 → Sonnet 확인 |
| `complex` | 복합 판단 또는 외부 검증 필요 | Sonnet 정밀 판단 |

---

## 제1호: 신의료기술 광고 금지

**탐지 난이도:** `complex`

**정규식 패턴:**
```regex
신(의료)?기술|새로운\s*치료(법|술)|혁신(적인)?\s*(치료|시술)|최신\s*(의료)?기술|첨단\s*치료|특허\s*(받은|출원)\s*(치료|시술)
```

**판단 기준:**
- 해당 기술이 제53조에 따른 신의료기술평가를 받았는지 확인 필요
- "최신", "혁신적" 표현만으로는 위반 아님 (신의료기술 여부 별도 확인)

**위반 예시:**
- "국내 최초 OO 줄기세포 치료법 도입"
- "미국 FDA 승인 신기술" (국내 미평가)

**비위반 예시:**
- "건강보험 적용 치료" (기존 인정 기술)
- "전문의 상담 후 적합한 치료법 안내"

## 제2호: 치료경험담 광고 금지

**탐지 난이도:** `context`

**정규식 패턴:**
```regex
(환자|고객)\s*(후기|리뷰|경험담|체험담|소감)|치료\s*경험담|전[·\-\s]?후\s*(비교|사진)|before\s*(and|&)?\s*after|시술\s*(전|후)|수술\s*(전|후)|(완치|완쾌|호전)\s*(되었|됐|했)|효과\s*(봤|있었|좋았)|만족(도|스럽|했)|추천(합니다|드려요|해요)|임상\s*경력\s*[1-6]\s*개월
```

**판단 기준:**
1. 환자 경험담이 치료 효과를 오인하게 할 우려가 있는지
2. 전후 사진이 치료 효과 홍보 목적인지
3. 6개월 이하 임상경력 광고 여부

**위반 예시:**
- "저는 이 병원에서 치료받고 완치되었습니다"
- "시술 전후 사진으로 놀라운 변화를 확인하세요"
- "수많은 환자분들이 효과를 경험하셨습니다"
- "개원 3개월 만에 100건 이상 시술"

**비위반 예시:**
- "상담 서비스가 친절했습니다" (치료 효과 무관)
- "주차가 편리해요" (시설 후기)

## 제3호: 거짓 광고 금지

**탐지 난이도:** `complex`

**정규식 패턴:**
```regex
(100|백)\s*%\s*(성공|완치|효과|안전)|부작용\s*(없|전혀|절대)|통증\s*(없|제로|zero)|무(조건|통증|흉터)|완벽(하게|한)|절대(적|로)\s*(안전|효과)
```

**판단 기준:**
- 객관적 사실과 다른 내용인지
- 검증 불가능한 절대적 표현인지
- 과학적/의학적 근거 유무

**위반 예시:**
- "100% 성공률 보장"
- "부작용이 전혀 없습니다"
- "국내 1위 병원" (검증 불가)
- "무통 시술" (통증 있을 수 있음)

**비위반 예시:**
- "마취 후 시술로 통증을 최소화합니다"
- "전문의 상담 후 진행됩니다"

## 제4호: 비교 광고 금지

**탐지 난이도:** `context`

**정규식 패턴:**
```regex
(다른|타)\s*(병원|의원|클리닉|의료기관).*보다|비교\s*불가|경쟁\s*병원|유일(하게|한)|독보적|최초|최고|1(위|등)|넘버원|No\.?\s*1|업계\s*(최초|유일|1위)
```

**판단 기준:**
- 다른 의료기관/의료인과 직접 비교 여부
- 암묵적 비교 (최고, 유일 등) 포함

**위반 예시:**
- "다른 병원보다 더 나은 결과"
- "업계 유일한 기술력"
- "지역 최고의 실력"
- "경쟁 병원과 비교해보세요"

**비위반 예시:**
- "경력 20년 전문의"
- "OO 대학병원 출신"

## 제5호: 비방 광고 금지

**탐지 난이도:** `context`

**정규식 패턴:**
```regex
(다른|타)\s*(병원|의원).*문제|경쟁\s*업체.*부작용|(저렴|싸구려).*병원|수준\s*낮|실력\s*없
```

**판단 기준:**
- 다른 의료인등을 비방할 목적이 있는지
- 불리한 사실을 광고하는지

**위반 예시:**
- "다른 병원에서 실패한 케이스를 저희가 해결"
- "저가 시술의 위험성을 아시나요?"
- "타 병원 재수술 전문"

**비위반 예시:**
- "재수술/교정 시술 가능" (타 병원 언급 없음)

## 제6호: 수술장면 노출 금지

**탐지 난이도:** `context`

**정규식 패턴:**
```regex
수술\s*(장면|영상|과정)|시술\s*(장면|영상|과정)|환부\s*(사진|영상)|출혈|절개|봉합|수술실\s*(공개|영상)|라이브\s*수술
```

**판단 기준:**
- 수술 장면, 환부 등이 포함되어 혐오감을 유발하는지
- 텍스트 광고에서는 해당 콘텐츠 존재 여부 암시 탐지

**위반 예시:**
- "실제 수술 과정 영상 공개"
- "시술 장면 라이브 중계"
- "환부 사진으로 확인하세요"

**비위반 예시:**
- "상담 과정 안내 영상"
- "시술 후 관리 방법 안내"

---

## 제7호: 부작용 정보 누락 금지

**탐지 난이도:** `complex`

**정규식 패턴:**
```regex
(부작용|주의사항|위험성)\s*(없|안전|걱정\s*없)|간단(하게|한)|쉽게|빠르게|금방|당일\s*(퇴원|일상)|회복\s*기간\s*(없|짧|빠른)|다운타임\s*(없|제로|최소)
```

**시술별 필수 고지 부작용:**

| 시술 | 필수 고지 부작용 |
|------|----------------|
| 보톡스 | 멍, 비대칭, 두통, 안검하수 |
| 필러 | 멍, 부기, 혈관 폐색, 괴사 |
| 리프팅 | 통증, 멍, 감각 이상, 비대칭 |
| 지방흡입 | 출혈, 감염, 피부 울퉁불퉁, 혈전 |
| 레이저 | 화상, 색소침착, 흉터 |
| 쌍꺼풀 | 부기, 비대칭, 흉터, 안구건조 |
| 코성형 | 부기, 감염, 비대칭, 호흡 장애 |
| 임플란트 | 감염, 골유착 실패, 신경 손상 |

**위반 예시:**
- "보톡스 - 간단하고 안전한 시술!" (부작용 미고지)
- "다운타임 제로! 바로 일상 복귀" (회복 기간 미고지)
- "무통, 무절개, 무흉터" (부작용 은폐)

**비위반 예시:**
- "시술 후 멍, 붓기가 있을 수 있으며 개인에 따라 차이가 있습니다"
- "부작용: 멍, 두통, 비대칭이 나타날 수 있습니다"

## 제8호: 과장 광고 금지

**탐지 난이도:** `complex`

**정규식 패턴:**
```regex
(국내|아시아|세계)\s*(최초|최고|유일|1위)|수천\s*(건|명|회)|압도적|탁월(한|하게)|독보적|혁신적|획기적|놀라운\s*(효과|결과)|기적(적|의)|드라마틱|마법(처럼|같은)|완벽(하게|한)\s*(해결|치료)
```

**판단 기준:**
- 객관적 사실을 과장하고 있는지
- 검증 불가능한 수치/표현인지

**위반 예시:**
- "수만 건의 시술 경험"
- "놀라운 효과를 경험하세요"
- "기적 같은 변화"
- "마법처럼 사라지는 주름"

**비위반 예시:**
- "10년 경력 전문의"
- "연간 1,000건 이상 시술" (검증 가능한 수치)

## 제9호: 무자격 표방 금지

**탐지 난이도:** `keyword`

**정규식 패턴:**
```regex
(인증|공인|자격)\s*(받은|취득|보유)|OO\s*전문(의|가)|명의|대한민국\s*대표|슈퍼닥터|베스트닥터|마스터|스페셜리스트
```

**판단 기준:**
- 법적 근거 있는 자격인지
- 법정 전문의 자격 외 명칭 사용 여부

**위반 예시:**
- "보톡스 전문의" (법적 자격 아님)
- "대한민국 대표 성형외과"
- "슈퍼닥터 선정"
- "코성형 마스터"

**비위반 예시:**
- "성형외과 전문의" (법정 자격)
- "피부과 전문의"
- "의학박사"

## 제10호: 기사형 광고 금지

**탐지 난이도:** `keyword`

**정규식 패턴:**
```regex
(기사|보도|뉴스|인터뷰)\s*(제공|협찬)|전문가\s*(의견|소견|추천)|의학\s*(칼럼|기고)|취재|리포트|특집\s*기사
```

**판단 기준:**
- 기사/전문가 의견 형식 + 연락처/약도 포함 여부
- 광고 표시 명확성

**위반 예시:**
- "[건강 칼럼] OO 치료법의 효과 - 문의: 02-XXX-XXXX"
- "[전문가 인터뷰] 명의가 말하는... (병원 주소 포함)"

**비위반 예시:**
- "[광고] 표시가 명확한 기사형 콘텐츠
- 연락처 없는 순수 정보성 기사

---

## 제11호: 미심의 광고 금지

**탐지 난이도:** `keyword`

**정규식 패턴:**
```regex
(심의|승인)\s*(번호|필|완료)|사전\s*심의|광고\s*심의
```

**판단 기준:**
- 심의 대상 매체인지 (신문, 인터넷신문, 정기간행물, 옥외광고물 등)
- 심의 번호 존재 여부

**심의 대상 매체:**
- 신문, 인터넷신문, 정기간행물
- 현수막, 벽보, 전단
- 전광판
- 일평균 이용자 10만명 이상 인터넷 매체
- SNS (일평균 이용자 10만명 이상)

**위반 예시:**
- 심의 번호 없는 옥외 광고
- 심의 내용과 다른 문구 사용

**비위반 예시:**
- "의료광고심의필 제2024-XXXX호"
- 의료기관 자체 블로그/홈페이지 (심의 면제)

## 제12호: 외국인환자 유치 국내광고 금지

**탐지 난이도:** `keyword`

**정규식 패턴:**
```regex
외국인\s*(환자|고객)\s*(유치|모집)|메디컬\s*투어|의료\s*관광|medical\s*tour(ism)?|해외\s*(환자|고객)|국제\s*진료
```

**위반 예시:**
- "외국인 환자 전문 클리닉"
- "Medical Tourism Welcome"
- "해외 환자 상담 가능"

**비위반 예시:**
- 해외 매체에 게재된 광고
- 외국어 홈페이지 (해외 접속 대상)

## 제13호: 비급여 할인 허위광고 금지

**탐지 난이도:** `context`

**정규식 패턴:**
```regex
할인\s*(이벤트|행사|특가)|[0-9]+%\s*(할인|세일|특가)|무료\s*(시술|상담|진료)|이벤트\s*가격|특별\s*가격|한정\s*할인|선착순\s*할인|오픈\s*기념|원가|정가\s*대비
```

**판단 기준:**
- 할인 금액, 대상, 기간, 범위 명시 여부
- 원가(정가) 정보의 진위 여부

**위반 예시:**
- "50% 할인 이벤트!" (기간, 대상 미명시)
- "정가 100만원 → 특가 50만원" (정가 허위 가능성)
- "무료 시술" (조건 미명시)

**비위반 예시:**
- "2024년 3월 한정, 신규 고객 대상 10% 할인 (상세 조건 문의)"
- "상담료 무료 (시술비 별도)"

---

## 제14호: 비공인 인증 광고 금지

**탐지 난이도:** `context`

**정규식 패턴:**
```regex
(수상|상장|감사장|표창)|인증\s*(받|획득|취득)|보증\s*(받|서)|추천\s*(병원|의원|클리닉)|베스트\s*(병원|의원|클리닉)|OO\s*어워드|OO\s*대상|(선정|선발)\s*(되|됨)
```

**공인 인증 예외:**
1. 제58조 의료기관 인증
2. 중앙행정기관, 지방자치단체, 공공기관 인증
3. 다른 법령에 따른 인증
4. WHO 협력 국제평가기구 인증

**위반 예시:**
- "2024 대한민국 베스트 병원 선정" (비공인 기관)
- "OO 협회 인증 병원" (법적 근거 없음)
- "고객 만족도 1위 수상"

**비위반 예시:**
- "의료기관 인증 획득" (제58조)
- "보건복지부 지정 전문병원"
- "JCI 인증 (WHO 협력 기구)"

## 제15호: 포괄적 금지 조항

**탐지 난이도:** `complex`

**정규식 패턴:**
```regex
(무조건|반드시|꼭)\s*(오세요|방문|예약)|(급|긴급)\s*모집|한정\s*수량|조기\s*마감|서두르세요|품절\s*임박|자리\s*없어요
```

**판단 기준:**
- 국민 보건에 해로운 영향
- 의료 질서를 해치는 행위
- 소비자 피해 우려
